# FOR-GOT 🧠
**Never forget a thing again.** FOR-GOT is a high-performance, offline-first Progressive Web App (PWA) designed for seamless task management. Built with Vanilla JavaScript and Firebase, it ensures your reminders are always with you—even without an internet connection.

## 🚀 Features
* **Offline-First:** Works without internet thanks to Service Worker caching.
* **Cloud Sync:** Automatically syncs your local reminders to Firebase Firestore when you're back online.
* **Google Authentication:** Secure login to keep your reminders private and accessible across devices.
* **Smart Notifications:** Triggered precisely at the scheduled time using the Web Notification API.
* **Mobile-Ready:** Fully responsive UI designed to be "Installed" as a standalone app.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Firebase Auth & Firestore
* **PWA:** Web Manifest & Service Workers
* **Hosting:** Vercel

## 📦 Installation
1. Clone the repo: `git clone https://github.com/saran1219/For-Got.git`
2. Add your `firebaseConfig` to `app.js`.
